Dynamic HTML Guide

This guide explains how to maintain the Pickles website using a template-based approach with pure HTML files. This approach provides a consistent look and feel across all pages without requiring PHP or JavaScript.

Note: This approach follows the user requirement of having no scripts while still maintaining a consistent design across all pages.

How the Template System Works

The website uses two template files:

  • header-template.html - Contains the standard HTML header, meta tags, CSS links, and navigation
  • footer-template.html - Contains the standard footer, copyright information, and closing HTML tags

When creating or updating a page, you'll copy content from these templates into your HTML file and customize the page-specific elements.

Creating a New Page

1 Start with the header template

Copy the content from header-template.html into your new page.

Update the following elements:

  • Page title: Change PAGE_TITLE to your specific page title
  • Meta description: Change PAGE_DESCRIPTION to your specific page description
  • Active navigation: Add the active class to the appropriate navigation link

2 Add your page-specific content

Add your main content between the header and footer sections:

<main class="section your-page-section">
    <!-- Your page content here -->
</main>

3 End with the footer template

Copy the content from footer-template.html at the end of your page.

Updating the Header or Footer

When you need to make a site-wide change to the header or footer:

1 Update the template files

Make your changes to header-template.html or footer-template.html.

2 Update all HTML pages

Copy the updated template content to all HTML pages on the site.

You can use find-and-replace in your code editor to make this process faster.

Example: Page Title and Description

For each page, customize these elements in the head section:

<title>Your Page Title — Pickles Friend Group</title>
<meta name="description" content="Your page-specific description goes here.">

Example: Active Navigation

To highlight the current page in the navigation menu, add the active class to the appropriate link:

<li><a href="index.html" class="active">Home</a></li>

Design System

Remember to maintain consistency with the established design system:

  • Colors: Primary (#305921), Secondary (#4CAF50), Accent (#8bc34a)
  • Typography: Open Sans (body), Playfair Display (headings)
  • Icons: Font Awesome icons for visual enhancement
  • Layout: Responsive design using CSS Grid and Flexbox
Important: Remember that this approach requires manual updates to all pages when making site-wide changes. While it requires more maintenance than a PHP-based solution, it allows the site to be hosted without a PHP server.
